ランチャーについての注意事項
画面が1920x1080以上に対応していないと崩れる場合があります。
通常だとTimer(Server)が起動していないとランチャーは接続待機中になります。
サーバーに繋げないでランチャーを開くには設定(launcher.ini)から
network = falseにしてください

操作方法
基本的にZで進む
ネットワーク通信を使用している場合時間になったら自動的にlauncherが終了画面に移行します。
そのさい、Bキーをキーボードより押すか15秒待つことによってトップ画面に戻ります。
launcherと終了したいときはescでは終了できません。Esc+X+Vで終了できます。

注意
右下のWindowsメニューから集中モードをオフにしてください。
ファイヤーウォールの許可設定が出てきた場合PrivateとPublicどちらも許可を選択してください。
PC番号は01~12のどれかを選択してください。

Launcherフォルダーのlauncher.iniで以下の設定ができます
・ネットワーク通信を使用するか
・PC番号を聞くか
・PC番号を設定
・BGMを流すか
・FPSを表示するか
・最初のロゴを表示するか

その他質問があれば何でも言ってください。
